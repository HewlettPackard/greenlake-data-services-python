## Notes for Large OpenAPI documents
If the OpenAPI document is large, imports in greenlake_data_services.apis and greenlake_data_services.models may fail with a
RecursionError indicating the maximum recursion limit has been exceeded. In that case, there are a couple of solutions:

Solution 1:
Use specific imports for apis and models like:
- `from greenlake_data_services.api.default_api import DefaultApi`
- `from greenlake_data_services.model.pet import Pet`

Solution 2:
Before importing the package, adjust the maximum recursion limit as shown below:
```
import sys
sys.setrecursionlimit(1500)
import greenlake_data_services
from greenlake_data_services.apis import *
from greenlake_data_services.models import *
```
